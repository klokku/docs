---
title: MacOS Menubar App
description: How to get and use the Klokku Menubar App for MacOS 
---

There is an official Klokku App available for MacOS.

This app allows you to check and change your current budget item using the MacOS menubar.

![Klokku-MacOS-screenshot.jpg](img/Klokku-MacOS-screenshot.jpg)

You can download the app here:
https://klokku.com/apps/klokku-macos.zip
